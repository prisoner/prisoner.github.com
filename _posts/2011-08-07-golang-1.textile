h2. 什么是Go语言?

bq. Go 编程语言是一个使得程序员更加有效率的开源项目。Go 是有表达力、简
洁、清晰和有效率的。它的并行机制使其很容易编写多核和网络应用,而新奇
的类型系统允许构建有弹性的模块化程序。Go 编译到机器码非常快速,同时
具有便利的垃圾回收和强大的运行时反射。它是快速的、静态类型编译语言,
但是感觉上是动态类型的,解释型语言。

以上解释来自Go官方网站, 由 "mikespook":http://www.mikespook.com/learning-go/ 翻译.

h2. 安装Go

  __以下安装过程在ubuntu 11.04中进行__

h3. 安装必需工具和库

<code>sudo apt-get install bison gawk gcc libc6-dev make</code>

h3. 加入环境变量

加入如下环境变量
<code>export GOROOT=/path/go/install</code>
<code>export GOOS=linux</code>
<code>export GOARCH=386</code>

  __GOROOT是源代码所在位置, 默认情况也是Go的安装位置. GOOS是操作系统代码. GOARCH是系统架构.__

取值表如下:

|_.GOOS|_.GOARCH|_.|
|darwin|  386| |
|darwin|  amd64| |
|freebsd| 386| |
|freebsd| amd64| |
|linux| 386| |
|linux| amd64| |
|linux| arm| incomplete |
|windows| 386| incomplete |

__具体的环境变量设置, 请参照 "这里":http://golang.org/doc/install.html#environment

h3. 获取并编译源代码

由于Go使用Mercurial管理源代码, 所以我们要先安装Mercurial來取得源代码

<code>
sudo apt-get install mercurial
</code>

clone源代码

<code>hg clone -u release https://go.googlecode.com/hg/ $GOROOT</code>

编译源代码

<code>cd $GOROOT/src</code>
<code>./all.bash</code>

编译完成出现<code>ALL TESTS PASSED</code>就表示安装成功了.

h2. 保持代码更新

取得最新的release, 可以运行以下命令

<code>cd go/src</code>
<code>hg pull</code>
<code>hg update release</code>
<code>./all.bash</code>
